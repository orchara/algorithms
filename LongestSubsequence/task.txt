Дано целое число 1 ≤ n ≤ 10^3 и массив A[1…n] натуральных чисел, не превосходящих 2⋅10^9. Выведите максимальное 1 ≤ k ≤n, для которого найдётся подпоследовательность 1 ≤ i_1 < i_2 <…< i_k ≤ n длины k, в которой каждый элемент делится на предыдущий 
(формально: для всех 1 ≤ j < k, A[i_j]|A[i_{j+ 1}]).

Sample Input:

4
3 6 7 12
Sample Output:

3
